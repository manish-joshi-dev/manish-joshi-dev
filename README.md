### 🚀 Featured Projects

#### 🦀 [Flow Tasks](https://github.com/joshi232004manish/flow-tasks) — Rust-native Android Task Manager
> A fully Rust-based Android app with **zero Java/Kotlin**. Slint UI + Axum embedded server + SQLx/SQLite, compiled directly to AArch64.

- Most "Rust on Android" apps call Rust from Kotlin — this one has no host language at all
- Migrated backend from Node.js to Axum for lower memory footprint
